# Hospital Bed & OT Utilization Analytics

## Overview
This project predicts hospital bed occupancy and OT utilization using SQL Server, Python, Azure Data Factory, Machine Learning, and Power BI.

## Features
- ICU occupancy monitoring
- OT utilization tracking
- SQL-based KPI calculations
- Forecasting hospital bed demand
- Power BI dashboards
- Automated ADF data refresh

## Technologies Used
- SQL Server
- Python
- Azure Data Factory
- Power BI
- Prophet Forecasting
- Jupyter Notebook

## KPIs Calculated
- Bed Turnover Ratio
- Average Length of Stay
- OT Utilization Percentage
- Forecasted Bed Demand

## Future Improvements
- Real-time hospital alerts
- SMS notifications for bed shortages
- AI-driven patient inflow forecasting
- Mobile healthcare dashboard
